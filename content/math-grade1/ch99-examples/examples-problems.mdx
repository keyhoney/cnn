# Problem — 6가지 유형 전체 예제

이 페이지는 `<Problem>` 컴포넌트의 **모든 유형**을 실제로 풀어볼 수 있게 구성했습니다.

<Callout type="info">
각 문제의 `id`는 **전체 교재에서 유일**해야 합니다. 예제는 `ex-prob-` 접두사를 사용했습니다.
</Callout>

---

## 1. mcq — 객관식

`answer`는 **0부터 시작** (첫 보기 = 0).

<Problem
  id="ex-prob-mcq-001"
  type="mcq"
  question="다음 중 $a^m \times a^n$ 을 간단히 한 결과는?"
  options={["$a^{m+n}$", "$a^{mn}$", "$a^{m-n}$", "$(a^m)^n$"]}
  answer={0}
  hints={[
    "같은 밑의 거듭제곱을 **곱할** 때 쓰는 법칙입니다.",
    "지수끼리 **더합니다**.",
    "$a^m \\times a^n = a^{m+n}$",
  ]}
  tags={["지수법칙", "곱셈"]}
  solution={[
    { step: "법칙 확인", content: "곱셈: $a^m \\times a^n = a^{m+n}$" },
    { step: "적용", content: "지수 $m$과 $n$을 더한다." },
    { step: "답", content: "$a^{m+n}$" },
  ]}
/>

## 2. numerical — 숫자 입력

`answer`에 숫자, `tolerance`로 오차 허용.

<Problem
  id="ex-prob-num-001"
  type="numerical"
  question="$3^2 \\times 3^3$ 의 값을 구하시오."
  answer={243}
  tolerance={0}
  timeLimit={120}
  hints={[
    "밑이 3으로 같습니다.",
    "$3^2 \\times 3^3 = 3^{2+3}$",
    "$3^5 = 243$",
  ]}
  tags={["지수법칙"]}
  solution={[
    { step: "식 정리", content: "$3^2 \\times 3^3 = 3^{2+3} = 3^5$" },
    { step: "계산", content: "$3^5 = 243$" },
  ]}
/>

### tolerance 예 (소수)

<Problem
  id="ex-prob-num-002"
  type="numerical"
  question="$\sqrt{2}$ 의 값을 소수 셋째 자리까지 구하시오. (반올림)"
  answer={1.414}
  tolerance={0.001}
  tags={["근호"]}
/>

---

## 3. formula — 수식(LaTeX) 입력

`accept`로 다른 표기도 정답 처리 가능.

<Problem
  id="ex-prob-formula-001"
  type="formula"
  question="$a^3 \\times a^4$ 를 간단히 하시오."
  answer="a^7"
  accept={["a^{7}"]}
  hints={["지수를 더하세요."]}
  tags={["지수법칙"]}
  solution={[
    { step: "지수 더하기", content: "$3 + 4 = 7$" },
    { step: "답", content: "$a^7$" },
  ]}
/>

---

## 4. fillblank — 빈칸 채우기

`question`에 `___` (언더스코어 3개), `blanks`에 순서대로 정답.

<Problem
  id="ex-prob-blank-001"
  type="fillblank"
  question="$a^m ___ a^n = a^{___}$ (곱셈)"
  blanks={["\\times", "m+n"]}
  tags={["지수법칙"]}
/>

<Problem
  id="ex-prob-blank-002"
  type="fillblank"
  question="$\\dfrac{a^m}{a^n} = a^{___}$"
  blanks={["m-n"]}
  tags={["지수법칙", "나눗셈"]}
/>

---

## 5. truefalse — 참·거짓

`answer={true}` 또는 `answer={false}`

<Problem
  id="ex-prob-tf-001"
  type="truefalse"
  question="$a \\neq 0$ 일 때 $a^0 = 1$ 이다."
  answer={true}
  tags={["지수"]}
/>

<Problem
  id="ex-prob-tf-002"
  type="truefalse"
  question="모든 실수 $a$에 대해 $a^2 > 0$ 이다."
  answer={false}
  hints={["$a = 0$ 일 때를 생각해 보세요."]}
  tags={["지수"]}
/>

## 6. order — 순서 배열

`items`는 **올바른 순서**. 화면에서는 섞여서 나옵니다.

<Problem
  id="ex-prob-order-001"
  type="order"
  question="같은 밑의 거듭제곱 곱셈을 지수의 합으로 나타내는 과정을 순서대로 배열하시오."
  items={[
    "밑이 같은 두 거듭제곱의 곱을 확인한다.",
    "지수끼리 더한다.",
    "결과를 $a^{m+n}$ 형태로 쓴다.",
  ]}
  tags={["지수법칙"]}
/>

<Problem
  id="ex-prob-order-002"
  type="order"
  question="$2^5 \\div 2^2$ 를 계산하는 과정을 순서대로 배열하시오."
  items={[
    "밑이 2로 같음을 확인한다.",
    "$2^{5-2} = 2^3$ 으로 쓴다.",
    "$2^3 = 8$ 을 구한다.",
  ]}
  tags={["지수법칙", "나눗셈"]}
/>
