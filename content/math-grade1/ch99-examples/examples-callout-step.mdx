# Callout · StepByStep · ImageNote · Table 예제

## 1. Callout — 5가지 유형

### definition (정의)

<Callout type="definition">
**지수**란 같은 수 $a$를 $n$번 곱한 것을 $a^n$으로 나타낸다. 이때 $a$를 **밑**, $n$을 **지수**라 한다.
</Callout>

### formula (공식)

<Callout type="formula">
같은 밑의 거듭제곱 곱셈: $a^m \times a^n = a^{m+n}$ $(a \neq 0)$
</Callout>

### warning (주의)

<Callout type="warning">
$0^0$ 과 $0^{-n}$ ($n > 0$) 은 **정의하지 않습니다.** 밑이 0인 경우 지수법칙을 적용할 때 특히 주의하세요.
</Callout>

### tip (팁)

<Callout type="tip">
지수법칙을 외울 때: **곱셈 → 지수 더하기**, **나눗셈 → 지수 빼기**로 기억하면 됩니다.
</Callout>

### info (참고)

<Callout type="info">
유리수·실수 지수는 **밑 $a > 0$** 일 때 자유롭게 사용할 수 있습니다.
</Callout>

## 5. 조합 예시

<Callout type="definition">
**로그** $\log_a b$ 는 $a^x = b$ 를 만족하는 $x$ 이다. ($a > 0$, $a \neq 1$, $b > 0$)
</Callout>

<StepByStep
  title="$\\log_2 8$ 구하기"
  steps={[
    { step: "정의", content: "$2^x = 8$ 을 만족하는 $x$ 를 찾는다." },
    { step: "지수로 바꾸기", content: "$8 = 2^3$ 이므로 $2^x = 2^3$" },
    { step: "답", content: "$x = 3$, 즉 $\\log_2 8 = 3$" },
  ]}
/>
