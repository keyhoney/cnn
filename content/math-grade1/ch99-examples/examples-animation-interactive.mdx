# Animation · Interactive 예제

## 1. Animation — 개념 애니메이션

미리 제작된 SVG 애니메이션입니다. `animationId`는 **아래 두 가지만** 사용할 수 있습니다.

| animationId | 내용 |
|-------------|------|
| `unit-circle` | 단위원과 $\sin\theta$, $\cos\theta$ |
| `quadratic-transform` | 이차함수 그래프의 평행이동·확대 |

### 단위원 (삼각함수)

<Animation animationId="unit-circle" title="단위원 (삼각함수)" speed={1} />

<Callout type="info">
재생·일시정지·속도 조절 버튼이 애니메이션 아래에 있습니다. `speed={1}` 이 기본 속도입니다.
</Callout>

### 이차함수 변환

<Animation
  animationId="quadratic-transform"
  title="이차함수 그래프 변환"
  speed={1}
/>

## 2. Interactive — 슬라이더 탐구

학생이 슬라이더로 계수를 바꾸며 **그래프와 수식**을 동시에 확인합니다.

### 이차함수 (3개 슬라이더)

<Interactive
  title="이차함수 그래프"
  formula="a*x^2 + b*x + c"
  formulaLatex="y = {a}x^2 + {b}x + {c}"
  params={[
    { name: "a", label: "a (이차항)", min: -2, max: 2, default: 1, step: 0.1 },
    { name: "b", label: "b (일차항)", min: -5, max: 5, default: 0, step: 0.1 },
    { name: "c", label: "c (상수항)", min: -5, max: 5, default: 0, step: 0.1 },
  ]}
  linkedGraph
  domain={[-4, 4]}
  range={[-6, 10]}
/>

### 일차함수 (2개 슬라이더)

<Interactive
  title="일차함수 y = ax + b"
  formula="a*x + b"
  formulaLatex="y = {a}x + {b}"
  params={[
    { name: "a", label: "기울기 a", min: -3, max: 3, default: 1, step: 0.1 },
    { name: "b", label: "y절편 b", min: -5, max: 5, default: 0, step: 0.1 },
  ]}
  linkedGraph
  domain={[-5, 5]}
  range={[-8, 8]}
/>

### 지수함수 (밑 a)

<Interactive
  title="지수함수 y = a^x"
  formula="a^x"
  formulaLatex="y = {a}^x"
  params={[
    { name: "a", label: "밑 a", min: 0.2, max: 4, default: 2, step: 0.1 },
  ]}
  linkedGraph
  domain={[-3, 3]}
  range={[-1, 10]}
/>

## 3. 속성 정리

### Animation

| 속성 | 필수 | 설명 |
|------|------|------|
| `animationId` | ✅ | `unit-circle` 또는 `quadratic-transform` |
| `title` | | 제목 |
| `speed` | | 재생 속도 (기본 1) |

### Interactive

| 속성 | 필수 | 설명 |
|------|------|------|
| `formula` | ✅ | function-plot 식 (`a`, `b` 등 변수명) |
| `params` | ✅ | 슬라이더 배열. `name`이 `formula`의 변수와 일치해야 함 |
| `formulaLatex` | | 화면 표시용. `{a}` 형태로 변수 연결 |
| `linkedGraph` | | `true`면 그래프 표시 |
| `domain` | | x축 범위 `[최소, 최대]` |
| `range` | | y축 범위 |
| `title` | | 패널 제목 |

<Callout type="warning">
`params`의 `name`과 `formula` 안 변수 이름이 **다르면** 슬라이더가 동작하지 않습니다. 예: `formula="a*x^2"` → `name: "a"`
</Callout>
