## 문제 2

$3^2 \times 3^3$ 의 값을 구하시오.

<Problem
  id="prob-002"
  type="numerical"
  question="$3^2 \\times 3^3$ 의 값을 구하시오."
  answer={243}
  tolerance={0}
  timeLimit={120}
  hints={[
    "밑이 3으로 같으므로 지수법칙을 적용할 수 있습니다.",
    "먼저 $3^2 \\times 3^3 = 3^{2+3}$ 으로 나타내세요.",
    "$3^5 = 243$ 입니다.",
  ]}
  tags={["지수법칙"]}
  solution={[
    { step: "식 정리", content: "$3^2 \\times 3^3 = 3^{2+3}$" },
    { step: "지수 계산", content: "$2 + 3 = 5$ 이므로 $3^5$ 입니다." },
    { step: "값 구하기", content: "$3^5 = 3 \\times 3 \\times 3 \\times 3 \\times 3 = 243$" },
  ]}
/>

## 문제 3

$\dfrac{2^7}{2^3}$ 을 간단히 하시오.

<Problem
  id="prob-003"
  type="numerical"
  question="$2^7 \\div 2^3$ 의 값을 구하시오."
  answer={16}
  tolerance={0}
  timeLimit={90}
  hints={[
    "나눗셈은 지수끼리 빼는 것으로 바꿀 수 있습니다.",
    "$2^7 \\div 2^3 = 2^{7-3}$",
    "$2^4 = 16$",
  ]}
  tags={["지수법칙", "나눗셈"]}
  solution={[
    { step: "나눗셈 법칙", content: "$a^m \\div a^n = a^{m-n}$ (단, $a \\neq 0$)" },
    { step: "지수 빼기", content: "$2^7 \\div 2^3 = 2^{7-3} = 2^4$" },
    { step: "계산", content: "$2^4 = 16$" },
  ]}
/>

<Callout type="tip">
곱셈은 지수를 더하고, 나눗셈은 지수를 빼면 됩니다.
</Callout>
